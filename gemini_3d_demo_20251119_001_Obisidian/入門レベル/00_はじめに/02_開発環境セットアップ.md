# 開発環境セットアップ

> [!abstract] 概要
> 学習を始めるために必要な開発環境をセットアップします。

---

## 必要なツール

以下のツールをインストールします：

| ツール | 用途 | 必須 |
|--------|------|------|
| Node.js | JavaScriptの実行環境 | ✅ |
| npm | パッケージ管理ツール | ✅（Node.jsに付属） |
| VS Code | テキストエディタ | 推奨 |
| Chrome | ブラウザ | 推奨 |
| Git | バージョン管理 | 任意 |

---

## 1. Node.js のインストール

### Node.js とは？

> [!info] Node.js
> **Node.js** は、ブラウザの外でJavaScriptを実行するための環境です。
> 開発サーバーの起動やパッケージ管理に使います。

### インストール手順（Windows）

1. [Node.js公式サイト](https://nodejs.org/) にアクセス
2. **LTS版**（推奨版）をダウンロード
3. ダウンロードしたインストーラーを実行
4. 画面の指示に従ってインストール

### インストール確認

コマンドプロンプト（またはPowerShell）を開いて、以下を実行：

```bash
node --version
```

バージョン番号が表示されればOKです：

```
v20.x.x
```

npm も確認：

```bash
npm --version
```

```
10.x.x
```

> [!warning] バージョンについて
> Node.js 18以上、npm 8以上を推奨します。

## 2. VS Code のインストール

### VS Code とは？

> [!info] VS Code
> **Visual Studio Code** は、Microsoftが開発した無料のテキストエディタです。
> プログラミングに特化した機能が豊富です。

### インストール手順

1. [VS Code公式サイト](https://code.visualstudio.com/) にアクセス
2. 「Download for Windows」をクリック
3. ダウンロードしたインストーラーを実行
4. 画面の指示に従ってインストール

### おすすめ拡張機能

VS Codeを起動したら、以下の拡張機能をインストールしましょう：

| 拡張機能 | 用途 |
|---------|------|
| Japanese Language Pack | 日本語化 |
| ESLint | コードチェック |
| Prettier | コード整形 |
| Live Server | ローカルサーバー |

**インストール方法：**
1. VS Code左側の拡張機能アイコン（四角が4つ）をクリック
2. 検索ボックスに拡張機能名を入力
3. 「インストール」をクリック

---

## 3. Chrome のインストール

### なぜ Chrome？

> [!info] 開発者ツール
> Chromeには強力な**開発者ツール**が付属しています。
> JavaScriptのデバッグやパフォーマンス分析に役立ちます。

### インストール手順

1. [Chrome公式サイト](https://www.google.com/chrome/) にアクセス
2. 「Chromeをダウンロード」をクリック
3. ダウンロードしたインストーラーを実行

### 開発者ツールの開き方

- **F12** キーを押す
- または **Ctrl + Shift + I**
- または 右クリック → 「検証」

---

## 4. プロジェクトのセットアップ

### プロジェクトフォルダを開く

1. VS Codeを起動
2. 「ファイル」→「フォルダーを開く」
3. プロジェクトフォルダを選択

```
C:\source\gemini_3d_demo_20251119_001
```

### 依存パッケージのインストール

VS Codeでターミナルを開きます：

- **Ctrl + `**（バッククォート）
- または「ターミナル」→「新しいターミナル」

以下のコマンドを実行：

```bash
npm install
```

> [!info] npm install
> `package.json` に記載された依存パッケージをダウンロードします。
> `node_modules` フォルダが作成されます。

成功すると以下のような表示が出ます：

```
added 123 packages in 10s
```

---

## 5. 開発サーバーの起動

### 起動コマンド

ターミナルで以下を実行：

```bash
npm run dev
```

成功すると以下のような表示が出ます：

```
  VITE v5.0.0  ready in 300 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

### ブラウザで確認

Chromeで以下のURLを開きます：

```
http://localhost:5173/
```

ゲームが表示されれば成功です！

### サーバーの停止

ターミナルで **Ctrl + C** を押すと停止します。

## よくあるトラブル

### npm install でエラーが出る

> [!warning] 対処法
> 1. Node.jsのバージョンを確認（18以上推奨）
> 2. `node_modules` フォルダを削除して再実行
> 3. 管理者権限でコマンドプロンプトを実行

```bash
# node_modules を削除して再インストール
rm -rf node_modules
npm install
```

### ポートが使用中

```
Error: Port 5173 is already in use
```

> [!warning] 対処法
> 別のターミナルで開発サーバーが動いている可能性があります。
> そのターミナルを閉じるか、別のポートで起動します。

```bash
npm run dev -- --port 3000
```

### ゲームが表示されない

> [!warning] 対処法
> 1. ブラウザのコンソール（F12）でエラーを確認
> 2. URLが正しいか確認（`http://localhost:5173/`）
> 3. サーバーが起動しているか確認

## フォルダ構成の確認

プロジェクトフォルダの構成を確認しましょう：

```
gemini_3d_demo_20251119_001/
├── index.html          ← エントリーポイント
├── package.json        ← プロジェクト設定
├── src/                ← ソースコード
│   ├── main.js
│   ├── Game.js
│   └── ...
├── public/             ← 静的ファイル
│   └── assets/
└── node_modules/       ← 依存パッケージ（自動生成）
```

> [!tip] node_modules について
> `node_modules` は `npm install` で自動生成されます。
> 中身を編集する必要はありません。
