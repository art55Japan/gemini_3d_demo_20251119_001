# 入門ドキュメント計画（改訂版）

## 概要

**プロジェクト名**: フェルトうさぎパラディン（Gemini 3D Demo）
**目的**: プログラミング入門者が、実際の3Dゲームのソースコードを読みながら、必要な知識を段階的に学ぶ実践的な入門書を作成する。

---

## コンセプト

1. **実践ファースト**: 実際のコードを読みながら、必要な知識をその都度学ぶ
2. **段階的学習**: 簡単なファイルから複雑なファイルへ進む
3. **完全解説**: コードの1行1行を丁寧に説明する
4. **手を動かす**: 値を変えて実験し、理解を深める
5. **マルチパラダイム**: 複数のプログラミングパラダイムを横断的に学ぶ

---

## このプロジェクトで学べるパラダイム

| パラダイム | 説明 | 主な登場箇所 |
|-----------|------|-------------|
| **オブジェクト指向（OOP）** | クラス、継承、ポリモーフィズム | Entity.js, Player.js |
| **関数型（FP）** | 高階関数、filter/map/forEach | Game.js, WorldManager.js |
| **イベント駆動** | イベントリスナー、コールバック | Input.js, SaveLoadUI.js |
| **ステートパターン** | 状態遷移、状態マシン | Slime.js, PlayerCombat.js |
| **コンポーネントベース** | 責務分割、疎結合 | Player.js（Physics/Combat/Collision） |
| **データ駆動** | 設定オブジェクト、マッピング | Input.js（keyMap）, Block.js（colorMap） |

## パラダイム別学習パス

読者の興味に応じて、パラダイム別に学習できるルートを提供します。

### オブジェクト指向を深く学びたい場合

```
Entity.js → Player.js → Slime.js → SaveManager.js
```

### 関数型プログラミングを学びたい場合

```
Game.js（配列操作） → EntityManager.js → PlayerPhysics.js → WorldManager.js
```

### ステートパターンを学びたい場合

```
PlayerCombat.js → Slime.js
```

### イベント駆動を学びたい場合

```
Input.js → SaveLoadUI.js → BuildSystem.js
```
